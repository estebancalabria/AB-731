# Copilot

Microsoft Copilot es una plataforma de IA basada en modelos de lenguaje (LLM) que se integra en Microsoft 365 y Azure para transformar el trabajo empresarial, automatizando tareas, generando información contextual y mejorando la toma de decisiones mediante el uso seguro de datos organizacionales a través de Microsoft Graph.

---

# 🧠 1. Concepto general

Copilot no es una única herramienta, sino un ecosistema de experiencias de IA:

* **Copilot Chat** → asistente conversacional para consultas rápidas y resúmenes con contexto organizacional.
* **Microsoft 365 Copilot** → integrado en Word, Excel, PowerPoint, Outlook y Teams para automatizar tareas diarias.
* **Copilot Studio** → plataforma low-code para crear copilotos personalizados.
* **Agentes especializados**:

  * **Investigador** → recopila y sintetiza información de múltiples fuentes.
  * **Analista** → transforma datos en insights, modelos y visualizaciones.

---

# 🔗 2. Microsoft Graph (base del sistema)

Microsoft Graph es el núcleo de contexto de Copilot:

* Conecta correo, archivos, chats, reuniones y calendarios.
* Permite respuestas basadas en datos reales de la organización.
* Respeta identidad, permisos y seguridad.
* Elimina silos de información.

👉 Resultado: Copilot entiende el “contexto real del negocio”.

# 💳 9. Licencias de Copilot

## 📦 Modelos de licencia de Copilot

* **Incluido con Microsoft 365** → implementación simple si ya tienes licencias compatibles.
* **Suscripción mensual** → costos predecibles y adopción continua.
* **Pago por uso** → ideal para pilotos o pruebas.

👉 Elección depende de escala, presupuesto y madurez.

---

## 🧾 Licencias base de Microsoft 365

* **E3** → productividad estándar y seguridad básica.
* **E5** → seguridad avanzada, cumplimiento y análisis.
* **Business Standard** → SMB con colaboración básica.
* **Business Premium** → SMB con seguridad avanzada.

👉 Copilot depende de estas licencias base.

---

## 🧠 Copilot Studio y costos adicionales

* Licenciamiento separado.
* Puede requerir consumo de Azure.
* Necesario para personalización avanzada e integraciones externas.

---

## 📊 Escenarios típicos

* Empresa grande (E5) → adopción incluida.

# 🧾 RESUMEN FINAL GLOBAL

Microsoft Copilot es una plataforma de inteligencia artificial integrada en Microsoft 365 y Azure que utiliza Microsoft Graph para conectar datos organizacionales y habilitar asistentes inteligentes en aplicaciones como Word, Excel, Teams y Outlook. Permite automatizar procesos documentales, analíticos y de reuniones, mejorar la toma de decisiones y optimizar la productividad, mientras se adapta mediante Copilot Studio y Azure AI, bajo un marco de inteligencia artificial responsable y distintos modelos de licenciamiento que van desde uso incluido hasta pago por uso y personalización avanzada.
